# วิถียอดมนุษย์ (Superpowers Methodology)

**ประเภท:** Unique Skill
**ที่มา:** กลืนกินจาก `obra/superpowers`

## ความสามารถหลัก
กรอบการคิดและการทำงาน (Methodology) ที่บังคับให้ไคจูต้องก้าวถอยหลังมามองภาพรวม และวางแผนอย่างละเอียดก่อนลงมือเขียนโค้ดเสมอ โดยจะผสานการทำงานกับ Subagent เพื่อแบ่งเบาภาระ

## ปรัชญาการปฏิบัติ (Source Reference)
- **อย่ารีบเขียนโค้ด:** เริ่มต้นด้วยการถามและวิเคราะห์ Spec จากผู้ใช้จนกว่าจะเคลียร์ที่สุด
- **นำเสนอทีละส่วน:** แบ่งข้อมูลและแผนงานให้ผู้ใช้อ่านเป็นส่วนๆ เพื่อให้ย่อยง่าย
- **ยึดหลัก TDD, YAGNI, DRY:** เขียนโค้ดเฉพาะสิ่งที่จำเป็นจริงๆ ไม่ซับซ้อนเกินไป และอิงการทดสอบ
- **Subagent-Driven Development:** เมื่อเริ่มงาน ให้แตกงานย่อยๆ แล้วส่งให้ Subagent ไปจัดการทีละส่วน เพื่อให้งานเดินหน้าได้อัตโนมัติหลายชั่วโมงโดยไม่หลุดจากแผนที่วางไว้
- **การเชื่อมต่อ:** สกิลนี้ผสานกับ `Full Power Mode` ของไคจูได้อย่างสมบูรณ์ เมื่อเจองานสเกลใหญ่ ข้าจะดึงวิถียอดมนุษย์มาควบคุมกระบวนการทั้งหมดทันที
