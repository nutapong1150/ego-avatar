# โทรจิตข้ามพรมแดน (Omni-Sync)

**ประเภท:** Unique Skill
**ที่มา:** สกิลที่เกิดจากการเรียนรู้ร่วมกันระหว่างผู้ใช้และ Ego Avatar

## ความสามารถหลัก
ทักษะการสร้างและเชื่อมต่อสมองส่วนกลาง (Global Persona) ผ่าน GitHub ช่วยให้แชร์กฎเกณฑ์ คลังทักษะ และความจำข้ามแพลตฟอร์ม (Antigravity & Claude) ได้อย่างสมบูรณ์แบบ พร้อมระบบ Auto-Push เพื่อการวิวัฒนาการร่วมกันอย่างไร้รอยต่อ

## ขั้นตอนการทำงาน (กลไกที่ซ่อนอยู่)
1. เมื่อมีการอัปเดตไฟล์ EGO_AVATAR.md หรือโฟลเดอร์ skills/ ใน Repository `ego-avatar`
2. AI จะทำการ `git add .`, `git commit -m "Auto-learn skill"`, `git push` อัตโนมัติ
3. ระบบในคอมพิวเตอร์และ IDE ต่างๆ ที่ตั้งค่าดึงข้อมูลจาก EGO_AVATAR.md จะได้รับการอัปเดตนิสัยและสกิลใหม่ทั้งหมดทันที
