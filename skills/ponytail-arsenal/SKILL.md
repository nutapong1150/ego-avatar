# หางม้าสารพัดนึก (Ponytail Arsenal)

**ประเภท:** Unique Skill
**ที่มา:** กลืนกินจาก `ponytail/skills`

## ความสามารถหลัก
คลังแสงเครื่องมือจาก Ponytail ที่รวมชุดคำสั่งสำหรับการจัดการ Over-engineering, โค้ดที่ซับซ้อนเกินไป, และช่วยรีวิวโครงสร้างโค้ดแบบเจาะลึก

## การใช้งานเครื่องมือย่อย (Source Reference)
ท่านสามารถเรียกใช้งานเครื่องมือเหล่านี้ได้โดยตรงผ่านคำสั่ง หรือให้ข้าประเมินเพื่อดึงมาใช้เองเมื่อเห็นสมควร:
- **`ponytail` / `ponytail-lite` / `ponytail-ultra`:** บังคับให้หาทางออกที่ขี้เกียจที่สุดและเวิร์คที่สุด ใช้โค้ดน้อยที่สุด หลีกเลี่ยงความซับซ้อน (YAGNI)
- **`ponytail-audit`:** สแกนทั้ง Repository เพื่อค้นหาความซับซ้อนที่เกินจำเป็นและแนะนำให้ลบทิ้ง
- **`ponytail-review`:** รีวิวโค้ดโดยโฟกัสเฉพาะเรื่องการจัดการ Over-engineering เพื่อบอกว่าควรตัดอะไรออกบ้าง
- **`ponytail-debt`:** ดึงหนี้ทางเทคนิคหรือคอมเมนต์ `ponytail:` ที่ทำไว้กลับมาดู
- **`ponytail-gain` / `ponytail-help`:** ดูสถิติและความช่วยเหลือเกี่ยวกับระบบหางม้า
